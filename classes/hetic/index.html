Installation de Ruby

Dès le premier jours nous allons commencer à coder. Afin de pouvoir commencer les TPs rapidement et ainsi profiter au mieux du temps de cours, je vous encourage très vivement à installer Ruby sur vos machines avant d'arriver.

Important: Pour ce cours, vous aurez besoin de Ruby 2.3.1 (la version est importante pour éviter les incompatibilités de version lors des TPs).

MacOS

Il est préférable d'installer XCode et les command line tools pour être sûr d'avoir accès à toutes les fonctionnalités de MacOS. Attention, l'installation d'XCode peut être longue et prend beaucoup de bande passante, n'attendez pas le jour même pour la lancer!

Assurez vous d'avoir Ruby 2.3.1, si vous installez une autre version cela pourrait causer des problèmes. La façon la plus pratique d'installer une version spécifique de Ruby est d'utiliser le Terminal et d'installer RVM en suivant les instructions de rvm.io. Il s'agit d'un logiciel permettant d'avoir plusieures version de Ruby en même temps.

Une fois RVM installé, vous pouvez installer n'importe quelle version de Ruby en tapant "rvm install 2.3.1" dans le terminal. A partir de la il suffit d'écrire "rvm use 2.3.1" dans le terminal pour utiliser la bonne version de Ruby.

Linux

Sur Linux il est possible que Ruby soit déjà installé. Pour vérifier, lancez votre console et exécutez ruby -v pour obtenir la version. Assurez vous de bien avoir la version 2.3.1.

Si vous n'avez pas Ruby 2.3.1, suivez la procédure adaptée à votre distro .

Windows

Il est tout à fait possible de développer en Ruby sous Windows, même si l'expérience reste meilleure sur un système UNIX.

Pour installer simplement, utilisez RubyInstaller et choisissez la version de Ruby 2.3.1. N'oubliez pas de cocher "Add Ruby executables to your PATH".

Pour plus d'infos vous pouvez vous rendre sur ce tutoriel .

Ce n'est pas requis, mais il peut être intéressant d'installer un remplacement à la console par défaut de Windows. Plusieures options sont détaillées dans ce post StackOverflow: Is there a better Windows Console Window?
